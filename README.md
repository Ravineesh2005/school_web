# K P Singh Intermediate College - Official Website

Welcome to the official project repository for **K P Singh Intermediate College (श्री महामाया शिक्षा निकेतन)** located in Arjunpur, Pratapgarh, Uttar Pradesh, India. This project is a comprehensive and responsive web presence designed to provide students, parents, and alumni with essential information about the school's facilities, academics, and culture.

## 🏫 About the Institution
Established in 1998, K P Singh Intermediate College provides world-class education with state-of-the-art facilities. The school's holistic approach nurtures both academic excellence and personal growth, preparing students for the modern technological landscape.

## ✨ Key Features
The website includes multiple pages, each dedicated to an aspect of the school's ecosystem:
- **Home (`index.html`)**: Overview of the school, its mission, core values, transport fleets, green initiatives, and labs.
- **Admission (`admission.html`)**: Details about the enrollment process and guidelines.
- **Academics (`academics.html`)**: Information regarding the curriculum and academic structure.
- **Students (`students.html`)**: Resources and details for current scholars.
- **Alumni (`alumni.html`)**: A portal celebrating past students and their success stories.
- **Teachers (`teachers.html`)**: A directory of our dedicated faculty members.
- **Events (`events.html`)**: Updates, galleries, and stories of school functions, sports, and cultural activities.
- **Contact Us (`contact-us.html`)**: Communication portals and location details.

## 💻 Technology Stack
This application is built using lightweight, foundational web technologies emphasizing speed and aesthetic design:
- **HTML5** for accessible, semantic structuring.
- **CSS3** for styling, responsive grid/flexbox layouts, dynamic micro-animations, and glassmorphic overlays.
- **Vanilla JavaScript** for interactive elements such as responsive navbars, custom alerts, and dynamic image transitions.
- **FontAwesome** for high-quality, scalable vector icons.

## 📂 Project Structure
For clarity and maintainability, styling and logic are largely separated on a per-page basis:
- `index.html`, `style.css`, `script.js` - Core homepage layout and functionality.
- Separate `.html`, `.css`, and `.js` file pairings for individual sections (e.g., `events.html`, `eventspagestyle.css`, `eventspagescript.js`).
- Shared module files for global components (`navbarstyle.css`, `titlebarstyle.css`, `alert.css`, `alert.js`).
- `assets/` - Directory holding all media elements like logos, gallery photos, and teacher/alumni portraits.
